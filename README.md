<!-- # 🌦️ Weather Ninja

**Weather Ninja** is a sleek, responsive, and user-friendly weather forecasting web application built with **HTML**, **CSS**, and **JavaScript**. It leverages the **OpenWeatherMap API** to provide real-time weather data for any city worldwide, presented with dynamic weather icons and a modern, visually appealing interface.

---

## 🚀 Features

- 🌍 **Real-Time Weather Data**: Fetches current weather information for any city using the OpenWeatherMap API.
- 🌤️ **Dynamic Weather Icons**: Displays custom icons for various conditions (Clear, Clouds, Rain, Snow, Thunderstorm).
- 🔎 **City Search**: Allows users to query weather for any city worldwide.
- 📱 **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- 💡 **Modern UI**: Features glassmorphism effects for a sleek, contemporary design.
- ⚠️ **Error Handling**: Includes input validation and user-friendly error messages.

## 🔑 Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.).
- An API key from [OpenWeatherMap](https://openweathermap.org/api).
- Optional: [VS Code Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for local development.

### Steps
1. **Clone the Repository**
   ```bash
   git clone https://github.com/Feyaz71/WEATHER-NINJA.git
   cd weather-ninja
   ```

2. **Add Your API Key**
   - Sign up at [OpenWeatherMap](https://openweathermap.org/) to obtain an API key.
   - Open `script.js` and replace the placeholder with your API key:
     ```javascript
     const apiKey = 'YOUR_API_KEY_HERE';
     ```

3. **Run Locally**
   - Open `index.html` in a browser, or
   - Use the **Live Server** extension in VS Code for a live preview.
