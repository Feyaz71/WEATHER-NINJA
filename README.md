## ðŸ”‘ Setup Instructions

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.).
- An API key from [OpenWeatherMap](https://openweathermap.org/api).
- Optional: [VS Code Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for local development.

### Steps
1. **Clone the Repository**
   ```bash
   git clone https://github.com/Feyaz71/WEATHER-NINJA.git
   cd weather-ninja
   ```

2. **Add Your API Key**
   - Sign up at [OpenWeatherMap](https://openweathermap.org/) to obtain an API key.
   - Open `script.js` and replace the placeholder with your API key:
     ```javascript
     const apiKey = 'YOUR_API_KEY_HERE';
     ```

3. **Run Locally**
   - Open `index.html` in a browser, or
   - Use the **Live Server** extension in VS Code for a live preview.
